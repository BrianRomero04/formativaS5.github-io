Introducción

Durante el primer cómputo de desarrollo web, adquirimos conocimientos sobre cómo se crean y funcionan las páginas web. Comenzamos entendiendo la estructura básica con HTML, CSS y JavaScript. Luego, aprendimos a utilizar Bootstrap para crear diseños responsivos y hacer las páginas más dinámicas.

Temas

  • Conceptos básicos de desarrollo web
  • Uso de Bootstrap y JavaScript
  • Estructuras y funciones en programación
  • Manipulación del DOM con JavaScript
  • WebComponents para componentes reutilizables
